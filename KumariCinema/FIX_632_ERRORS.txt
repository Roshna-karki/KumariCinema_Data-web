TROUBLESHOOTING - IF PACKAGE MANAGER DOESN'T WORK
????????????????????????????????????????????????????????????????????????????????

Issue: "Unable to find package source"
Fix:   - Check internet connection
       - Check if NuGet.org is accessible
       - Try: Tools > Options > NuGet > Package Sources (ensure nuget.org is enabled)

Issue: "Package not found for .NET Framework 4.7.2"
Fix:   - Version 21.11.0 supports .NET Framework 4.7.2 ?
       - Try an older version: 19.18.0

Issue: "Access denied"
Fix:   - Might need admin mode in Visual Studio
       - Run Visual Studio as Administrator
       - Retry NuGet install

Issue: "EULA not accepted"
Fix:   - Accept the license in the NuGet dialog
       - Or add -AcceptLicense flag in PowerShell
