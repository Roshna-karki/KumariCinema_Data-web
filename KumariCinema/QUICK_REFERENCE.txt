?? QUICK SQL QUERIES
????????????????????????????????????????????????????????????????????????????????

Get Total Movies:
  SELECT COUNT(*) FROM Movie;

Get All Theatres:
  SELECT TheatreID, TheatreName, TheatreCity FROM Theatre;

Get Today's Shows:
  SELECT * FROM Shows WHERE TRUNC(ShowDate) = TRUNC(SYSDATE);

Get Confirmed Bookings:
  SELECT * FROM Booking WHERE BookingStatus = 'CONFIRMED';

Get Payment Summary:
  SELECT PaymentStatus, COUNT(*), SUM(PaymentAmount) 
  FROM Payment GROUP BY PaymentStatus;

Get Ticket Status:
  SELECT TicketStatus, COUNT(*) FROM Ticket GROUP BY TicketStatus;

Get Hall Capacity Info:
  SELECT h.HallName, h.HallCapacity, t.TheatreName 
  FROM Hall h INNER JOIN Theatre t ON h.TheatreID = t.TheatreID;

?? DEBUGGING TIPS
????????????????????????????????????????????????????????????????????????????????
Check Debug Output:
  - Visual Studio > View > Output Window
  - Look for System.Diagnostics.Debug.WriteLine() messages

Enable Error Details:
  - Web.config > <system.web> > <customErrors mode="Off" />
  - Shows detailed error messages

Database Issues:
  - Test connection with SQL*Plus
  - Verify user permissions
  - Check if tables exist: SELECT * FROM user_tables;

Application Errors:
  - Check browser console (F12)
  - Look for exception in Visual Studio output
  - Verify all files saved

?? TROUBLESHOOTING
????????????????????????????????????????????????????????????????????????????????

Problem: "Cannot connect to database"
Solution: 
  1. Check Oracle is running (Services)
  2. Verify connection string in Web.config
  3. Test with SQL*Plus

Problem: "Oracle.ManagedDataAccess not found"
Solution:
  1. Package Manager Console
  2. Install-Package Oracle.ManagedDataAccess

Problem: "No tables found"
Solution:
  1. Run DATABASE_SETUP.sql
  2. Verify as CINEMA user
  3. Check user_tables: SELECT * FROM user_tables;

Problem: "Charts not displaying"
Solution:
  1. Check browser console (F12)
  2. Verify Chart.js CDN accessible
  3. Check hidden field has data

Problem: "Pagination not working"
Solution:
  1. Check GridView PageSize = 20
  2. Check data source has records
  3. Verify Page_Load sets !IsPostBack
