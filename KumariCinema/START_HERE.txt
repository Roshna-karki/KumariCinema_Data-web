STEP 1: INSTALL ORACLE CLIENT LIBRARIES (3 minutes)
????????????????????????????????????????????????????????????????????????????????

Objective: Add Oracle database support to the project

ACTION:
  1. In Visual Studio, go to MENU > Tools > NuGet Package Manager > Package Manager Console

  2. You'll see a "Package Manager Console" window open at the bottom

  3. Copy this command:
     Install-Package Oracle.ManagedDataAccess

  4. Paste it in the Package Manager Console and press ENTER

  5. WAIT for the console to show "Successfully installed"
     (This takes 1-2 minutes, don't interrupt)

  6. After success, copy this second command:
     Install-Package Oracle.ManagedDataAccess.EntityFramework

  7. Paste and press ENTER

  8. WAIT for "Successfully installed" again

? DONE: Oracle libraries installed
  Check: In Solution Explorer, expand the "packages" folder
         You should see "Oracle.ManagedDataAccess" folder

TROUBLESHOOTING

If you see "Parser Error" or "Could not load type":
  ? You're on Step 1
  ? Have you installed Oracle.ManagedDataAccess?
  ? If yes, close and reopen Visual Studio
  ? Then try Build > Clean Solution > Build Solution again

If Application doesn't start:
  ? Check internet connection
  ? Check if another application is using port 80/443
  ? Check Windows Firewall settings
  ? Restart Visual Studio

If Database connection fails:
  ? Is Oracle Database running?
  ? Is connection string correct? (Web.config)
  ? Have you created the "cinema" user?
  ? Have you run DATABASE_SETUP.sql?
