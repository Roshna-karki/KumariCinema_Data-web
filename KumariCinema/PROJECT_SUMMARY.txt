CUSTOMIZATION GUIDE
================================================================================

TO ADD MORE MOVIE GENRES:
- Edit MovieDetails.aspx > ddlGenre DropDownList
- Add new <asp:ListItem> for each genre

TO ADD MORE LANGUAGES:
- Edit MovieDetails.aspx > ddlLanguage DropDownList
- Add new <asp:ListItem> for each language

TO ADD MORE PAYMENT METHODS:
- Edit PaymentDetails.aspx > ddlMethod DropDownList
- Add new <asp:ListItem> for each method

TO CHANGE COLOR SCHEME:
- AdminMaster.Master > <style> section
- Modify CSS variables in :root selector

TO ADD NEW FIELDS TO TABLES:
1. Alter table in Oracle:
   ALTER TABLE TableName ADD NewColumn DataType;
2. Add form field in .aspx page
3. Update .aspx.cs code to save/retrieve new field

TROUBLESHOOTING GUIDE
================================================================================

ISSUE: Application won't start
FIX: Check Web.config, ensure database is running, verify connection string

ISSUE: GridView won't load data
FIX: Check database tables exist, verify connection permissions

ISSUE: Pages show blank
FIX: Check Debug Output, verify all .aspx files are linked to correct .cs files

ISSUE: Charts not displaying
FIX: Verify Chart.js CDN is accessible, check browser console for errors

ISSUE: Forms won't save data
FIX: Check form validation, verify parameterized queries, check database permissions

SEE: SETUP_GUIDE.txt for detailed troubleshooting
